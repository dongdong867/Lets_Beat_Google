## **Build With** ##

FrontEnd:

* Nuxt3.js
* Vue3.js
* TypeScript
* axios
* ESLint
* Prettier

\
Backend:

* Java
* Spring Boot
* jsoup

***

## **Development Sever** ##

**Run Springboot_Project at first.**\
Then start the development sever by.

```bash
npm run dev
```

***

## **Production** ##

Build the application for production.

```bash
npm run build
```

Locally preview production build.

```bash
npm run preview
```
