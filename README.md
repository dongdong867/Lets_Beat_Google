## **Build With** ##
FrontEnd:
* Nuxt3.js
* Vue3.js
* TypeScript
* axios
* ESLint
* Prettier
<br>

Backend:
* Java
* Spring Boot
* jsoup
<br>
***

## **Development Sever** ##
**Run Springboot_Project at first.**<br>
Then start the development sever on http://localhost:3000.
```
npm run dev
```
***

## **Production** ##
Build the application for production.
```
npm run build
```
Locally preview production build.
```
npm run preview
```
